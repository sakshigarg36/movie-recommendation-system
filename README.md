# Movie Recommendation System

A movie recommendation system built using content-based filtering techniques that suggests similar movies based on user selection.

#Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

# Features

* Content-based movie recommendations
* Similarity score computation
* Interactive Streamlit interface
* User-friendly recommendation system

# How It Works

The system analyzes movie metadata and computes similarity scores to recommend movies related to the selected title.

